### 1. Авторизация (Login)

- [ ] **Позитивная проверка**: Вход с `standard_user` и паролем `secret_sauce` перенаправляет на страницу `/inventory.html`.

- [ ] **Негативная проверка**: Оставить поля "Username" и "Password" пустыми -> ошибка: *"Username is required"*.

- [ ] **Негативная проверка"**: Заполнить только пароль -> ошибка: *"Username is required"*.

- [ ] **Негативная проверка"**: Заполнить только логин -> ошибка: *"Password is required"*.

- [ ] **Негативная проверка**: Ввод невалидного логина/пароля -> ошибка: *"Username and password do not match"*.


### 2. Страница "Products" (Товары)

- [ ] **Отображение карточек**: Проверить, что на странице отображается 6 товаров (по умолчанию).
- [ ] **Данные товара**: У каждого товара есть название, цена и описание.
- [ ] **Изображения**: Проверить, что изображения товаров загружаются (особенно для `problem_user`).
- [ ] **Кнопка "Add to cart"**: Кнопка добавляет товар в корзину и меняет текст на "Remove".
- [ ] **Кнопка "Remove"**: Удаляет товар из корзины и меняет текст обратно на "Add to cart".
- [ ] **Бейдж корзины**: Счетчик на иконке корзины обновляется при добавлении/удалении товаров.

### 3. Сортировка (Sort)

- [ ] **Name (A to Z)**: Сортировка по алфавиту от А до Я (значение по умолчанию).
- [ ] **Name (Z to A)**: Сортировка по алфавиту от Я до А.
- [ ] **Price (low to high)**: Сортировка по цене от низкой к высокой.
- [ ] **Price (high to low)**: Сортировка по цене от высокой к низкой.

### 4. Корзина (Cart)

- [ ] **Добавление товара**: Товар, добавленный на главной, отображается в корзине.
- [ ] **Удаление из корзины**: Кнопка "Remove" в корзине удаляет товар.
- [ ] **Continue Shopping**: Кнопка возвращает на страницу со списком товаров (`/inventory.html`).
- [ ] **Checkout**: Кнопка "Checkout" ведет на страницу `/checkout-step-one.html`.

### 5. Оформление заказа (Checkout)

- [ ] **Шаг 1 (Your Information)**:
    - [ ] Успешное заполнение формы (First Name, Last Name, Zip/Postal Code) и переход на шаг 2.
    - [ ] Пустые поля: Проверка валидации на обязательность полей.
- [ ] **Шаг 2 (Overview)**:
    - [ ] Отображение добавленных товаров, их цены и итоговая сумма (Item total, Tax, Total).
    - [ ] Кнопка "Finish" завершает заказ.
    - [ ] Кнопка "Cancel" возвращает на главную страницу товаров.
- [ ] **Шаг 3 (Complete!)**:
    - [ ] Отображение сообщения об успешном заказе: *"Thank you for your order!"*.
    - [ ] Кнопка "Back Home" возвращает на главную страницу.

### 6. Меню (Боковое меню)

- [ ] **Открытие/Закрытие**: Клик по иконке "гамбургера" открывает и закрывает меню.
- [ ] **All Items**: Переход на страницу `/inventory.html` из любого раздела.
- [ ] **About**: Переход по внешней ссылке (обычно на сайт Sauce Labs).
- [ ] **Logout**: Разлогинивает пользователя и возвращает на страницу `/`.
- [ ] **Reset App State**: Очищает корзину, не разлогинивая пользователя.
